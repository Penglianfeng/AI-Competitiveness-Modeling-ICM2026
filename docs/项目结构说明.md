# 项目结构概览

本文档描述了重构后的项目目录结构。

## 主要目录

### `configs/` - 配置文件
- `config.py` - 全局配置，包含：
  - 项目路径配置
  - 目标国家列表
  - 分析年份范围
  - 模型超参数
  - 指标体系定义

### `src/` - 源代码
核心代码模块，采用包结构组织。

#### `src/data/` - 数据处理
- `loaders.py` - 统一的数据加载接口
- `scrapers/` - 数据爬虫脚本
  - `ai_talent_scraper.py` - AI人才数据 (World Bank, UNESCO)
  - `uis_rd_scraper.py` - UNESCO R&D数据
  - `world_bank_scraper.py` - 世界银行宏观指标

#### `src/models/` - 建模代码
- `problem1_2/` - 问题一&二解决方案
  - `solution.py` - 完整实现（DEMATEL-TOPSIS-Malmquist）
- `problem3/` - 问题三解决方案
  - `glv_forecast.py` - GLV-SD 分层生态演化模型（2026-2035预测）

#### `src/utils/` - 工具函数
- `preprocessing.py` - 数据预处理（Winsorization, 归一化等）
- `visualization.py` - 可视化工具

### `scripts/` - 运行入口
- `run_problem1_2.py` - 运行问题一&二求解
- `run_problem3.py` - 运行问题三预测
- `run_scrapers.py` - 运行数据爬虫

### `outputs/` - 输出结果
- `problem1_2/` - 问题一&二的结果文件
  - CSV数据文件（dematel_weights, topsis_scores等）
  - PNG图表文件
- `problem3/` - 问题三的结果文件
  - `glv_params_rK.csv` - GLV模型拟合参数
  - `glv_simulated_states_2026_2035.csv` - 状态预测结果
  - `topsis_forecast_scores_2026_2035.csv` - 竞争力预测排名
  - `forecast_trends.png` - 趋势图

### `docs/` - 文档
- 数据文件指标分类对应表.txt
- 文件目录介绍.txt

### `notebooks/` - Jupyter Notebooks
用于交互式分析（待添加）

## 数据目录（保持原位）

由于数据量较大且移动有风险，以下目录保持原位：
- `b题数据源/` - 主要原始数据
- `Research and development investment and innovation foundation/` - R&D数据
- `Supply, Mobility and Quality of AI Talents/` - 人才数据
- `问题一二结果/` - 原始结果备份

## 运行方式

```bash
# 安装依赖
pip install -r requirements.txt

# 运行问题1&2
python scripts/run_problem1_2.py

# 运行问题3（需先运行问题1&2生成数据）
python scripts/run_problem3.py

# 或使用模块方式
python -m src.models.problem1_2.solution
python -m src.models.problem3.glv_forecast
```
